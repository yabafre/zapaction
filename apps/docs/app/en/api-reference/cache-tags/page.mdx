# Cache Tags

Tags provide cache coherence across server and client.

```ts
const usersKeys = createFeatureKeys("users", {
  list: () => [],
});

setTagRegistry({
  users: [usersKeys.list()],
});
```

```ts
const createUser = defineAction({
  input: schema,
  output: outputSchema,
  tags: ["users"],
  handler: async ({ input }) => db.user.create({ data: input }),
});
```

- Server side: `revalidateTag`
- Client side: `invalidateQueries`
