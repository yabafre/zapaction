# Cache Tags Pattern

Tags drive invalidation across server and client.

```ts
export const usersTags = createFeatureTags("users", {
  list: () => [],
  detail: (id: string) => [id],
});
```

## Example

```ts
export const createUser = defineAction({
  input: schema,
  output: outputSchema,
  tags: [usersTags.list()],
  handler: async ({ input }) => db.user.create({ data: input }),
});
```

- Server revalidation: `revalidateTag`
- Client invalidation: tag registry -> `invalidateQueries`
