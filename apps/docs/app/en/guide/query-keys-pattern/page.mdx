# Query Keys Pattern

Use feature-scoped keys instead of global factories.

```ts
export const usersKeys = createFeatureKeys("users", {
  list: () => [],
  detail: (id: string) => [id],
});
```

## Output

- `usersKeys.list()` -> `["users"]`
- `usersKeys.detail("42")` -> `["users", "42"]`

Keys must stay JSON-serializable.
