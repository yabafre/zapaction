# What is ZapAction

ZapAction is a framework layer that keeps Next.js Server Actions and React Query in sync with a predictable contract.

## Core capabilities

- `defineAction` with typed input/output validation
- Tag-based cache coherence across server and client
- Feature-first architecture patterns for real projects
- Query hooks that stay aligned with server contracts

## 10-line flow

```ts
const usersKeys = createFeatureKeys("users", {
  list: () => [],
});

export const getUsers = defineAction({
  input: z.object({}),
  output: z.array(z.object({ id: z.string(), name: z.string() })),
  tags: ["users"],
  handler: async () => db.user.findMany(),
});
```

```tsx
const { data } = useActionQuery(getUsers, {
  input: {},
  queryKey: usersKeys.list(),
  readPolicy: "read-only",
});
```
