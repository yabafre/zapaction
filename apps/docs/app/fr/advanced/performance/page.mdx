# Performance

ZapAction se concentre sur un comportement de cache previsible.

## Ce que ca ameliore

- Moins de bugs d'invalidation manuelle
- Moins de sur-fetch lie a des refetch ad-hoc
- Repartition plus claire des responsabilites serveur/client

## Ce que ZapAction ne promet pas

- Pas de garanties de benchmark universelles
- Pas d'optimisation automatique de code reseau non lie
