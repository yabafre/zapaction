# Cache Tags

Les tags assurent la coherence de cache entre serveur et client.

```ts
const usersKeys = createFeatureKeys("users", {
  list: () => [],
});

setTagRegistry({
  users: [usersKeys.list()],
});
```

```ts
const createUser = defineAction({
  input: schema,
  output: outputSchema,
  tags: ["users"],
  handler: async ({ input }) => db.user.create({ data: input }),
});
```

- Cote serveur: `revalidateTag`
- Cote client: `invalidateQueries`
