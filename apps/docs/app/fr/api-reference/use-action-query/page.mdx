# useActionQuery

A utiliser uniquement pour les chemins de lecture.

```tsx
const query = useActionQuery(listUsers, {
  input: {},
  queryKey: usersKeys.list(),
  readPolicy: "read-only",
});
```

## Contrat

- `queryKey` explicite obligatoire
- `readPolicy: "read-only"` obligatoire
- Types input/output inferes depuis le contrat d'action
