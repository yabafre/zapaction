# Pourquoi ZapAction

ZapAction est optimise pour les equipes qui utilisent deja Next.js Server Actions et TanStack Query.

## Differences cles

| Sujet | Setup classique | ZapAction |
| --- | --- | --- |
| Contrat d'action | reparti dans plusieurs fichiers | un contrat type unique |
| Query keys | tableaux ad-hoc | helpers feature-scope |
| Invalidation | manuelle par mutation | tags + registry par defaut |
| Structure projet | dispersion par routes | modules feature-first |
