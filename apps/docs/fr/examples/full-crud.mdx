# Full CRUD Example

## Chemin repository

```txt workspace
examples/next-app/
```

## Ce que montre cet exemple

- Contrats d'action avec `defineAction`.
- Lectures client avec `useActionQuery`.
- Ecritures client avec `useActionMutation`.
- Boundaries d'erreur autour des composants client.
