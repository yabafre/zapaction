# Introduction

ZapAction est une couche de contrats types pour **Next.js Server Actions** et **TanStack Query**.

## Ce que tu obtiens

- Validation input/output avec `defineAction` et Zod
- Query keys et cache tags predictibles
- Invalidation de cache coherente serveur + client
- Parcours d'adoption clair pour les equipes

## Parcours recommande

1. Commencer par **Project Setup**
2. Continuer avec **React Query**
3. Verifier avec **Testing Patterns**
4. Utiliser **Full CRUD Example** comme reference
